# Modora Bot

A production-ready, multi-server ModMail bot for Discord.

## Phase 1

- Core structure created
- MongoDB connection established
- Bot can start and respond to `/ping`

## Setup

1. Copy `.env.example` to `.env` and fill in your `BOT_TOKEN` and `MONGO_URI`.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python main.py`
